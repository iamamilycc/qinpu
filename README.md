## 功能（Phase 1+2 已完成）
- **减字谱 → 简谱**：点选音色（散/按/泛）、弦、徽位徽分、左右手指法拼出减字，自动译出简谱音高
- **简谱 → 减字谱**：输入简谱旋律（含小节线/拍号/附点/八分组/延音/休止/八度点），自动编配指法（散音优先→常用把位→少换弦），点击减字可切换备选弹法
- **三行对照谱**：简谱行＋简化五线谱行（低音谱表 F 大调，自绘 SVG 零依赖）＋减字行，仿谱书排版
- **试听**：Karplus-Strong 物理弦模型模拟古琴拨弦（按纯律真实音高发声），谱面同步高亮
- **打印/存 PDF**：只印谱面
- **教程页**：减字结构、三种音色、右手八法、左手走音、徽位徽分、软件用法
- 减字用 SVG 动态组字渲染（Unicode 无减字谱字符）
- 内置示例：《普庵咒》《关山月》开头（正调 1=F）

## 文件结构
- `index.html` / `style.css` —— 页面与样式（含打印样式）
- `js/pitch.js` —— 音律引擎（正调定弦、徽位↔音高、自动编配；纯逻辑可单测）
- `js/jianzi.js` —— 减字 SVG 组字渲染
- `js/staff.js` —— 简化五线谱 SVG 渲染（谱号/调号/符桿/符杠/加线自绘）
- `js/audio.js` —— 模拟古琴音试听（Web Audio，零依赖）
- `js/app.js` —— 界面逻辑与教程注入
- `tests/test_pitch.js` —— 音律引擎单元测试：`node tests/test_pitch.js`（19 条）

## 已知限制（诚实清单）
1. 减字谱不记节奏 → 只转换音高，节奏须人工打谱
2. 指法编配是"一种"弹法，非唯一答案
3. 仅支持正调（1=F）；其他调弦待加
4. 部分减字省文字形（挑/剔/摘/擘等）待琴谱校对，暂以全字或近似部件显示
5. 五线谱为简化版：无三连音/延音线/装饰音记号；换行不重绘谱号
6. 试听为合成音色非真琴采样；泛音/走音（绰注吟猱）不影响发声

## 待外部条件（需要用户提供才能做）
- **拍照识别减字（OCR）**：需视觉大模型 API（如 Anthropic API Key）；识别后仍需逐字人工核对
- **真古琴音色**：需下载古琴采样音源（sf2/wav 采样包），替换合成音
- **对外发布**：GitHub Pages 建仓推送（用户点头后执行）

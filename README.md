# 琴谱通 —— 古琴减字谱 ⇄ 简谱双向转换

纯静态站（HTML/CSS/JS），无后端、无依赖、无需 API，可直接双击 `index.html` 打开，或部署到 GitHub Pages。

## 功能

### 双向转换
- **简谱 → 减字谱**：输入简谱旋律，自动编配古琴指法生成减字谱；点击任一减字可切换备选弹法，或自定义
- **减字谱 → 简谱**：点选音色（散/按/泛）、弦、徽位徽分、左右手指法拼出减字，自动译出简谱音高
- **文字减字谱解析**：直接贴「散勾一 / 名九挑四 / 大七六托五 / 上九」这类琴人口述格式，自动出谱

### 谱面
- **三行对照谱**：简谱行 ＋ 五线谱行 ＋ 减字行，仿谱书排版
- **减字 SVG 动态组字**（Unicode 无减字字符）：字形取自 GlyphWiki 专业字形，按《减字组字文法》五大律（容器/截取/叠合/框架/工艺）拼装
- **五线谱**：自绘 SVG 零依赖，谱号/调号（升降号皆按五度圈推导）/符杆/符杠/附点/加线/变音记号/连音线/三连音/休止符
- **竖排减字谱**：古籍样式，可单独打印
- **打印 / 存 PDF**：只印谱面

### 记谱法
小节线、反复（含一二房）、拍号、附点、八分/十六分组、三连音、延音线、休止符、
八度点、升降号、延长号、中途变速（`T=60`）、泛音段（`[泛 … ]泛`）

### 编配引擎
- **17 种调弦**全部支持，切换即重新编配：
  正调 1=F ／ 借正调 1=C ／ 蕤宾·紧五 1=♭B ／ 黄钟·紧五慢一 ／ 离忧·紧五慢一二 ／ 凄凉·紧二五 ／
  慢角·慢三 1=C ／ 玉女·慢一三 ／ 慢商·慢二 ／ 间弦·紧五慢三 ／ 无媒·慢三六 ／ 慢宫·慢一三六 1=G ／
  商调·慢一三四六 1=D ／ 侧商·慢三四六 ／ 碧玉·紧三慢一四六 ／ 清商·紧二五七 1=♭E ／ 捣衣·紧二五七慢一
- **编配画像**：宏（散音骨架）／圆（走韵悠扬）／远（泛音清冷）／琴歌·师承，切换即重编
- **依据琴学惯习**：上行绰·下行注、长音吟猱、散按相间、指法防单调轮转、人体力学换弦罚分、
  对句成对编配、文句法节奏、三连重复音散-按-散、长音八度撮
- 音高模型经国琴网正调按/泛音定弦校验表验证 **9/9 纯律吻合**

### 试听
- **真古琴采样**（Freesound #176266，CC-BY）按最近空弦变速取音，Karplus-Strong 物理弦模型兜底
- 指法音色分族（向内温厚／向外清亮／泛音如铃／按音略暗）、走音滑音、绰注吟猱撞逗等 22 种技法发声
- 谱面同步高亮、速度滑块、AB 小节循环、逐音跟弹

### 录入与工具
- **哼唱转谱**（`hum.js`）：麦克风哼一段，YIN 基频检测纯前端出谱，无需 API
- **MIDI 导入**（`midi.js`）：解析 SMF 单声部转简谱
- **拍谱识谱**（`ocr.js`）：拍照识别谱面（接智谱 GLM-4.6V-Flash 免费视觉模型或 Anthropic/OpenAI，Key 仅存本机）
- **本地曲库 / 分享链接**：谱子存浏览器，或编码进 URL 分享
- **教程页**：减字结构、三种音色、右手八法（含琴面俯视动作示意）、左手走音、徽位徽分、谱字术语

### 内置示例曲
普庵咒 ／ 关山月 ／ 高山·页1 ／ 秋风词（慢角 1=C）／ 极乐吟（紧五 1=♭B）／
湘妃怨（借正调 1=C）／ 捣衣（紧二五七慢一 1=♭E）／ 古怨（侧商调 1=D·宋·姜夔）

| 文件 | 作用 |
|---|---|
| `index.html` / `style.css` | 页面与样式（含打印样式） |

| `js/samples.js` | 内嵌古琴采样（base64） |
| `js/hum.js` / `js/midi.js` / `js/ocr.js` | 哼唱转谱 / MIDI 导入 / 拍谱识谱 |

| `scripts/scorecrop.py` | 谱图裁剪工具（网格标尺 / 墨迹投影切列 / 画框自检） |

## 已知限制（诚实清单）

1. **减字谱不记节奏** —— 减字只决定音高（弦＋徽算出），节奏须人工打谱；
   「打谱风格」引擎给的是机器演绎参考，**不等同于传承打谱**
2. **编配是「一种」弹法**，非唯一答案；多弦扫类（滚拂历泼剌索铃）与猱撞进复
   设计上不自动使用，避免替原曲编创
3. **内置名曲的节奏比例为近似，待校**；音高经逐音闭环核对（秋风词/极乐吟/湘妃怨/捣衣/古怨），
   节奏与原谱可能有出入
4. 古怨仅录 1=D 段首二行（原谱中段转 1=C，引擎一次一个记谱基准）
5. 少数减字字形（索铃/长锁/撮框架/绰点朝上）GlyphWiki 无收录，以部件拼装兜底
6. 拍谱识谱的真实识别准确率待更多实拍验证；识别后仍须逐字人工核对
7. 五线谱暂无倚音记号
